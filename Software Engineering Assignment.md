# **Assignment: Build a Mini Laundry Order Management System (AI-First)**

## **Objective**

Build a simple system that allows a dry cleaning store to:

* Create orders  
* Track order status  
* Calculate billing  
* View basic dashboard data

You are expected to **use AI tools (ChatGPT, Copilot, etc.) heavily** and demonstrate how you leverage them.

---

## **Time Limit**

**72 hours**

---

## **Problem Statement**

A dry cleaning store needs a lightweight system to manage daily orders.

### **Core Features**

#### **1\. Create Order**

* Customer Name  
* Phone Number  
* Garments (e.g., Shirt, Pants, Saree)  
* Quantity per garment  
* Price per item (you can hardcode or configure)

👉 Output:

* Total bill amount  
* Unique Order ID

---

#### **2\. Order Status Management**

Each order should have statuses:

* `RECEIVED`  
* `PROCESSING`  
* `READY`  
* `DELIVERED`

👉 Ability to update status

---

#### **3\. View Orders**

* List all orders  
* Filter by:  
  * Status  
  * Customer name / phone

---

#### **4\. Basic Dashboard**

Return:

* Total orders  
* Total revenue  
* Orders per status

---
